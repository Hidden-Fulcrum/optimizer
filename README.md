# Workflow Optimizer

A production workflow optimization tool for meal preparation and kitchen operations.

## Features

- Production quantity planning
- Task time estimation and optimization
- Production scenario management
- Production log tracking
- Inventory management

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Express + tRPC
- **Database**: PostgreSQL (Supabase)
- **ORM**: Drizzle ORM

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Set up environment variables:
   - Create a `.env` file with your `DATABASE_URL`

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open http://localhost:3000 in your browser
